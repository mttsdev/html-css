Trabalhando com listas

Listas ordenadas

Vamos criar uma rotina diária, como no exemplo:

  1. Acordar
  2. Tomar banho
  3. Tomar café da manhã
  4. Escovar os dentes
  5. Vou à faculdade
  6. Vou ao trabalho
  7. Chego em casa

Analisando o código acima, o ol serve para indicar uma lista de itens e o li é para indicar os itens que vão ser incluídos
Uma observação é que no li, a partir do HTML5, é opcional o </li> no final, mas vai no caso do desenvolvedor. mesmo caso do ul embaixo.

O parâmetro presente no ol é a personalização da lista, cada um com números normais ou romanas, ou letras.

Listas não ordenadas

  • Pão
  • Leite
  • Café
  • Feijão
  • Arroz

o ul é uma lista que não precisa de ordem numérica como "Faço isso primeiro, depois faço isso".
